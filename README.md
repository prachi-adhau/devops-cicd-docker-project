# CI/CD Pipeline for Dockerized Flask Application using GitHub Actions

## Project Overview
This project demonstrates a basic CI/CD pipeline using GitHub Actions and Docker.

## Objective
To automate the build, test, and deployment process of a containerized Flask application.

## Tools Used
- Python (Flask)
- Docker
- Docker Compose
- GitHub Actions
- GitHub
- Docker Hub

## Project Structure
devops-cicd-docker-project/
│── app.py
│── requirements.txt
│── Dockerfile
│── docker-compose.yml
│── .github/workflows/ci-cd.yml
│── README.md

## Planned Workflow
1. Create Flask app
2. Dockerize application
3. Configure GitHub Actions
4. Automate testing and image build
5. Push image to Docker Hub

## Status
Project setup initialized. CI/CD workflow and containerization structure prepared.
